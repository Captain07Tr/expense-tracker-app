# expense-tracker-app
💰 Income & Expense Tracker – Windows Forms

Bu proje, C# Windows Forms kullanılarak geliştirilmiş bir gelir ve gider takip uygulamasıdır.

---

📌 Proje Hakkında

Kullanıcıların günlük gelir ve giderlerini ekleyebildiği, takip edebildiği ve yönetebildiği masaüstü bir finans takip uygulamasıdır.

---

⚙️ Kullanılan Teknolojiler

- C#
- Windows Forms (.NET Framework)
- SQL (varsa belirt)

---

🚀 Özellikler

- Gelir ekleme sistemi
- Gider ekleme sistemi
- Kategori bazlı finans takibi
- Veri listeleme ve yönetme
- Basit ve kullanıcı dostu arayüz

---

🎯 Amaç

Bu proje, Windows Forms kullanarak kullanıcı etkileşimli bir finans yönetim sistemi geliştirmek ve veri yönetimi mantığını öğrenmek amacıyla yapılmıştır.
